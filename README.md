# Bridget Jones Diary App

The app allows users to create, view, update, and delete diary entries. Each entry includes a date, time, category, and text content. 

## Technologies Used

- Frontend: HTML, CSS
- Backend: Node.js, Express
- Database: SQL
- Architecture: MVC

## Set up

1. Clone the repo

```bash 
git clone <your-repo-link>
cd <project-folder>
```
2. Install dependencies

```bash 
npm install
npm i express cors dotenv fs pg
npm i -D nodemon
```
3. Set up the database

- Create a SQL database
- Update your database connection details
- Create .env file
- Add the database link + password, as well as PORT=3000 or one of your choice

4. Run the app
```bash 
npm start
```
5. Open in browser
```bash 
http://localhost:3000
```
